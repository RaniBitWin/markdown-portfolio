# Learning how to make headers in Markdown
- The header tag is a ``` # ```.
- Just like in html there are six sizes of headers.
- Each size is defined by the number of ``` # ``` that are placed at the beginning of it and separated by a space.
- As shown below:
# ``` # ``` The size 1. 
## ``` ## ``` The size 2.
### ``` ### ``` The size 3. 
#### ``` #### ``` The size 4. 
##### ``` ##### ``` The size 5.
###### ``` ###### ``` The size 6.
